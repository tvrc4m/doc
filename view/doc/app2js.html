app与js交互

  1. JS加载完毕之后，调用客户端方法，loadComplete()方法 目的：通知客户端加载已经完成
  2. JS需要提供一个方法setActivityState("state")
  3. 说明：此方法主要是客户端调用js，以告知js具体的状态，以下是状态说明（后期可能更改暂定这些）：
    1. 未登录，未分享
    2. 未登录，已分享
    3. 已登录，已分享，但是没有手机号
    4. 已登录，已分享，并且绑定手机号了
    5. 已登录，未分享(初步考虑，这个不会有，但是为了兼容以后，暂时写上)
    6. 没有抽奖机会
  4. 客户端需要提供一个带返回值的方法，getSessionId( return sessionId) 说明：供给js调用，随时获取sessionId
  5. 客户端需要提供一个方法 startClickPrize() 说明：JS通知客户端，用户已经点击了抽奖按钮
